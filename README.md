# MEGABOX Ad Sales Portfolio

메가박스 광고영업 직무 지원을 위해 만든 정적 웹 포트폴리오입니다.

## 파일 구성

- `index.html`: 메인 페이지
- `style.css`: 전체 스타일
- `script.js`: 스크롤 애니메이션, 카운터, 네비게이션 활성화

## 실행 방법

브라우저에서 `index.html`을 열면 바로 확인할 수 있습니다.

## GitHub Pages 배포 방법

1. 이 폴더를 GitHub 저장소에 업로드합니다.
2. GitHub 저장소의 `Settings > Pages`로 이동합니다.
3. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
4. 브랜치는 `main`, 폴더는 `/ (root)`를 선택합니다.
5. 저장 후 몇 분 뒤 배포 주소가 생성됩니다.

## 사이트 메시지 방향

이 사이트는 단순 자기소개보다 아래 메시지가 보이도록 설계했습니다.

- 메가박스의 상영 환경을 광고 상품으로 재해석할 수 있는 지원자
- 광고주의 과제를 데이터와 제안서로 풀어낼 수 있는 지원자
- 신규 광고주 발굴과 재집행 유도까지 고려하는 영업형 인재
